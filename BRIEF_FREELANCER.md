# 🍽️ Système POS Restaurant - Brief Freelancer

## 📋 Résumé Rapide

**Restaurant marocain** kayna 3andi, baghi n-installik système POS digital m3a:
- POS l'caisse (tablette)
- Kitchen display l'couzina
- App livreur
- Stock management automatic
- Multi-staff m3a roles (admin, cashier, chef, livreur)

**L'code kamel mawjoud déjà.** Khass-ni ghir freelancer y-deployi ou y-configurih.

---

## 🎯 Khdma Khassa

### **Phase 1: Setup Technique (4-6h)**

1. **Supabase setup**
   - 3andi déjà account ou project mawjoudin
   - Khass run SQL files (5 files) f'tartib
   - Setup Authentication
   - Configure RLS policies

2. **Deployment**
   - Deploy 5 HTML files 3la Netlify (free)
   - Configure domain (optionnel)
   - Test 3la mobile/tablette/desktop

3. **Testing**
   - Test login admin
   - Test POS → Kitchen → Livreur workflow
   - Verify stock deduction automatic
   - Verify realtime notifications

### **Phase 2: Personnalisation (2-4h)**

1. **Données réelles**
   - Import menu dyali (~30-50 plats)
   - Configure ingredients ou recettes
   - Add 10+ staff members b'roles

2. **Branding**
   - Logo dyal restaurant
   - Esm restaurant f'interface

### **Phase 3: Training (2-3h)**

1. Training session m3a staff
2. Document simple (PDF, 3-5 pages)
3. Support 7 yyam ba3d livraison

## 📦 Files Mawjoudin (13 fichiers)

### **SQL Files (5):**
- `restaurant_schema.sql` - 9 tables + triggers + sample data
- `modifiers_schema.sql` - Tables dyal modifiers
- `delivery_trigger.sql` - Auto-create delivery
- `auth_setup.sql` - Auth integration
- `rls_policies.sql` - Security policies

### **HTML Files (7):**
- `login.html` - Login unifié multi-rôles
- `admin_panel.html` - Gestion staff
- `pos_caisse_v2.html` - POS tablette
- `kitchen_display.html` - Kitchen display
- `app_livreur.html` - PWA livreur
- `test_supabase.html` - Connection test
- `pos_caisse.html` - Version 1 (archive)

### **Documentation (1):**
- `README.md` - Setup guide complet en darija

## 🛠️ Tech Stack

- **Backend:** Supabase (PostgreSQL + Auth + Realtime + Storage)
